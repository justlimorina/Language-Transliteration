Introducing the Azerbaijani/Turkish Non-Latin Special Characters Input Method

Overview

This guide introduces how to type Azerbaijani/Turkish characters using an Anglicized method, which makes use of Latin characters to represent non-Latin ones. Below is a complete guide on how to convert each character along with their corresponding IPA notation and pronunciation notes.

Example

The word Azərbaycan in Azerbaijani can be typed as Azaerbaycan using this method.

Similarly, Özgürlük can be typed as Oezuerguerluek, and İstanbul becomes Iistanbul.

For a name like Şəms, you would type Shaems, and Qarabağ would be written as Qarabagh.
